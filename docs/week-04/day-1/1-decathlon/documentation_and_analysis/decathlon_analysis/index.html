Decathlon Analysis

Who had the longest long jump seen in the data?

decathlon_analysis$long_jump_max <- decathlon_data %>% 
  filter(
    event_name == "Long Jump"
  ) %>% 
  select(
    name,
    event_score
  ) %>% 
  arrange(
    desc(event_score)
  ) %>% 
  head(1)
## # A tibble: 1 x 2
##   name  event_score
##   <chr>       <dbl>
## 1 Clay         7.96

The longest recorded long jump in this data set was achieved by Clay, with a distance of 7.96 m.

What was the average 100m time in each competition?

decathlon_analysis$average_100m <- decathlon_data %>% 
  filter(
    event_name == "Sprint, 100m"
  ) %>% 
  group_by(
    competition
  ) %>% 
  summarise(
    mean_100m_time = mean(event_score)
  ) %>% 
  mutate(
    mean_100m_time = round(mean_100m_time, 2)
  )
## # A tibble: 2 x 2
##   competition mean_100m_time
##   <chr>                <dbl>
## 1 Decastar              11.2
## 2 Olympics              10.9

The average 100 m sprint time in the Decastar was 11.18 s. The average 100 m sprint time in the Olympics was 10.92 s.

Who had the highest total points across both competitions?

decathlon_analysis$points <- decathlon_data %>% 
  select(
    name,
    competition,
    points
  ) %>% 
  distinct() %>% 
  group_by(
    name
  ) %>% 
  summarise(
    total_points = sum(points)
  ) %>% 
  arrange(
    desc(total_points)
  ) %>% 
  head(1)
## # A tibble: 1 x 2
##   name   total_points
##   <chr>         <dbl>
## 1 Sebrle        17110

The highest total points across both competitions was attained by Sebrle, with a total points score of 17,110.

What was the shot-put scores for the top three competitors in each competition?

Note that we are considering the top three competitors (assumed to mean in each competition, not each event) only.

decathlon_analysis$shot_put <- decathlon_data %>%

What was the average points for competitors who ran the 400m in less than 50 seconds vs.Â those than ran 400m in more than 50 seconds?

decathlon_analysis$fast_sprinter <- decathlon_data %>% 
  filter(
    event_name == "Sprint, 400m"
  ) %>% 
  mutate(
    fast_sprinter = ifelse(
      event_score < 50,
      TRUE,
      FALSE
    )
  ) %>% 
  group_by(
    fast_sprinter
  ) %>% 
  summarise(
    mean_points = mean(points)
  ) %>% 
  mutate(
    mean_points = round(mean_points, 2)
  ) %>% 
  arrange(
    desc(mean_points)
  )
## # A tibble: 2 x 2
##   fast_sprinter mean_points
##   <lgl>               <dbl>
## 1 TRUE                8120.
## 2 FALSE               7727.

The average (competition) points for an decathlete who completes the 400 m sprint in less than 50 seconds is 8120.48. The average points for an athlete who completes the 400 m sprint in greater than (or equal to) 50 seconds is 7727.17. This is a total difference of 393.31.

Predictably, decathletes who complete the 400 m sprint quickly have greater mean competition-wide points total.
