Question 1

If I roll a six-sided dice once, what is the probability of getting a number less than 5?

d6_lt5 <- rolldie(1) %>% 
  filter(X1 < 5)
nrow(d6_lt5)/nrow(
  rolldie(1)
)
## [1] 0.6666667

Question 3

Question 3a

How many outcomes are in the event total_equals_5_6_7 (i.e.Â the total of the four dice equals \(5\), \(6\) or \(7\))?

total_equals_5_6_7 <- rolldie(4) %>% 
  mutate(
    "sum" = X1 + X2 + X3 + X4
  ) %>% 
  filter(
    sum %in% c(5, 6, 7)
  )
nrow(total_equals_5_6_7)
## [1] 34

Question 3b

What is the probability of this event?

nrow(total_equals_5_6_7)/nrow(rolldie(4))
## [1] 0.02623457

Question 3c

What assumptions do we apply in answering part iii?

We assume that the observations are independent. We also assume that the dice are fair.

Question 4

  permutations(
    n = 5,
    r = 5
  )
)
## [1] 0.008333333
